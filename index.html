HTML (Hypertext Markup Language).

The basic structure of an HTML tag:
<tag name>contents</tag name>

Here are some tag examples:

  1. for Bold: b. Example: <b>Hello</b>. This will make the word "Hello" bold.
  2. for Italics: em.
  3. for insertion of hyperlink: <a href = "www.cnn.com">News</a>. When you click on "News", you will be taken to the website www.cnn.com.
  4. for images: <img scr="url">. The "url" here will contain the path to the image. No closing tag needed.
  5. Paragraph: <p>contents</p>

There is a difference between inline and block-level tags.

Lesson # 3 - Cascading Style Sheets (CSS)

Introduction to CSS:

CSS allows us to add style to a webpage. Int allows one to code repetition. Avoiding repetition is important because it saves time, minimizes coding errors and saves time debugging.

CSS is designed to enable separation of document content from presentation, including elements such as layout, colors and fonts.

CSS can be inserted directly into an HTML document using <style> tag or written in a separate document and using <link> tag in the HTML doc to reference it.


Inheritance:
  1. This is the mechanism by which properties are applied not only to a specified element but also to its descendants.
  2. In general, descendants inherit text properties but no box properties
  3. Some properties that can be inherited: color, font, letter-spacing, line-height, list-style, text-align and word-spacing.
  4. Properties that cannot be inherited: backgrond, border, display, height and width, overflow etc.
  5. Inheritance prevents certain properties from being declared repeatedly.

CSS Selectors:

Go to this page to more information on selectors: CSS Selectors

Selectors allow you to apply syling to specific HTML elements.


Types of Selectors:
  1. ID selector
  2. Class selector
  3. Tag selector
  4. Attribute selector
  5. Positional selector

3 Ways of Including CSS Styling in a Webpage:
  1. Write CSS in the <head> section of your HTML.
  2. Link your HTML to a separate CSS file.
  3. Write your style inline with your HTML. Not the prefered way!
